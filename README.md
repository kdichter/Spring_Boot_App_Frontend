# Contact Management Application (Frontend)

This repository contains the **React frontend** for the Contact Management Application.

The frontend provides the user interface for interacting with the backend REST API, allowing users to securely manage contacts through a web interface.

The application communicates with the Spring Boot backend and is containerized using Docker as part of the full application stack.

---

## Tech Stack

- React
- JavaScript
- HTML/CSS
- Docker

---
## Running the Application
This frontend is containerized and runs via Docker Compose in the backend repository.
Follow the instructions in the [backend repository](https://github.com/kdichter/Spring_Boot_App_Backend) to start the full application.
